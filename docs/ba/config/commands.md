## Player Commands

### Game Commands
* `/<game> join`
* `/<game> leave`

**Examples:**
* `/arena join`
* `/arena leave`

### Join Phase Commands
* `/<game> info`
* `/<game> result`
* `/<game> options`
  
### Team Commands
* `/team info`
* `/team decline`
* `/team disband`
* `/team join`
* `/team create <player 1> <player 2> <player x...>`
